# 💈 Plataforma Web de Productos de Barbería

## 📌 Descripción del Proyecto
Este proyecto consiste en una **plataforma web de venta online de productos de barbería para hombres**, diseñada para ofrecer una experiencia completa al usuario.

Además de la tienda, la plataforma incluye un sistema de **asesoría personalizada** que ayuda a los clientes a elegir los productos más adecuados según sus necesidades.

---

## 🎯 Objetivo
El objetivo principal es desarrollar una aplicación web funcional como práctica educativa, aplicando los conocimientos adquiridos en el ciclo formativo de Desarrollo de Aplicaciones Web.

---

## 🛠️ Tecnologías Utilizadas
- **Framework:** SvelteKit
- **Lenguaje principal:** JavaScript
- **Frontend:** HTML, CSS

### ¿Por qué SvelteKit?
SvelteKit es un framework moderno basado en **Svelte**, un compilador de interfaces de usuario que se diferencia de otros frameworks como React o Vue en que **no utiliza un DOM virtual**. En lugar de ello, compila los componentes a JavaScript puro en tiempo de build, lo que se traduce en aplicaciones más ligeras, rápidas y con menos código en el navegador.

Entre sus principales ventajas destacan:

- **Rendimiento superior:** al eliminar la capa del DOM virtual, las actualizaciones de la interfaz son más eficientes.
- **Sintaxis sencilla e intuitiva:** permite escribir componentes combinando HTML, CSS y JavaScript en un único fichero `.svelte`, reduciendo la curva de aprendizaje.
- **Enrutamiento integrado:** SvelteKit incluye un sistema de rutas basado en la estructura de carpetas del proyecto, sin necesidad de librerías externas.
- **Soporte SSR y SSG:** permite renderizado en el servidor (Server-Side Rendering) y generación de páginas estáticas (Static Site Generation) de forma nativa.
- **Ecosistema moderno:** integración sencilla con Vite como bundler, lo que agiliza el desarrollo con recarga en caliente (HMR) casi instantánea.
